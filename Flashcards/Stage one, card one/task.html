Description

A flashcard is a piece of paper that contains a term on one side and a related text, which may be translation or explanation, on the other. Flashcards are often used for learning a foreign language, so for this project, weâ€™ll say that the text on the back of the card is the definition of a term.

In this project, you will create an application that emulates a set of flashcards on a screen. While developing this application, you will not only learn some programming, but you will also save paper!

To begin with, implement a program that outputs a single term and its definition. At this stage, the term can be anything you want.

How to submit?

This stage is auto-graded. The grader will check that:

  • You output 4 lines.
  • The first line is Card:.
  • The third line is Definition:.

Examples

Here are some output examples to clarify what the result should look like.

Card:
purchase
Definition:
buy

Definition:
